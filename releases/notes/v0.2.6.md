# New Horizons Desktop v0.2.6

## Changes

### Pressure Calibration: Tare UI

- Added `calibration_dump_tare` and `calibration_capture_tare` entries to the
  terminal command list, enabling tare inspection and direct tare capture from
  the built-in terminal.
- Guarded pressure calibration against runaway: added a safety check that
  prevents the calibration client from continuing if the pressure reading
  exceeds a threshold unexpectedly.
- Fixed pressure calibration API alignment between frontend and backend route
  signatures.

### Battery: Charge Profile Rename

- Charge profiles renamed for clarity: `ultra_slow` / `slow` / `balanced` /
  `fast` / `extreme`.
- All five profiles are now shown in the Battery settings dropdown; default
  is `balanced`.
- Removed redundant Termination% and Precharge% metrics from the battery
  status card.
