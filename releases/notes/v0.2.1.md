# New Horizons Desktop v0.2.1

## Fixes

- **Correct `PressureCalClient` method names in API routes**: `pressure_target`
  renamed to `set_target`; `pressure_stop` renamed to `stop`.
- **Add `User-Agent` header to `PressureCalClient`**: requests without a
  `User-Agent` were blocked by Cloudflare bot detection on the lab server.
- **Start all presets from 3 kPa**: previously presets began at 0 kPa (below
  the bag pre-inflation point), which caused the first calibration step to fail.
  All presets now begin at 3 kPa.
- **Safety double-stop after session commit**: after `calibration_session_commit`,
  the client now explicitly sends `PSET,0` followed by `PCTRL,0` to prevent any
  residual pressurisation after the session ends.
- **Restore pressure calibration readings compatibility**: fixed a regression
  where live pressure readings were no longer displayed during the calibration
  workflow.
