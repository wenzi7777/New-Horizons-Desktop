# New Horizons Desktop v0.2.9

## Changes

### Pressure Calibration: Auto-Apply Range After Commit

- After completing a calibration session and committing, the calibration range is
  now automatically applied without requiring manual adjustment. Implemented by
  issuing an automatic `calibration_status` query immediately after
  `calibration_session_commit`; the frontend receives the updated `enabled: true`
  state and activates the range without user interaction.

### Pressure Calibration: "Apply Tare" Button in Maintenance UI

- Added an "Apply tare" button to the device Maintenance section (Settings →
  Maintenance). The button sends the `calibration_tare_capture` command, samples
  sensors for 1 second, and refreshes calibration status — all without requiring
  an active calibration session. Requires maintenance mode to be active.
- Added `applyTare`, `applyTareHint`, and `applyTareSuccess` i18n strings for
  English and Japanese locales.
- Added `calibration_tare_capture` to the device command allowlist.
