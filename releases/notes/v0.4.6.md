# New Horizons Desktop v0.4.6

## Fixes

### Launchpad folder layout refinement

- Reworked the Launchpad home layout so device groups render as a compact,
  scannable workspace instead of oversized empty panels.
- Updated folder cards to behave more like real entry points, with compact
  count badges, stable 2x2 previews, and disabled low-intensity empty states
  when a group has no devices.
- Tightened the Global Apps section so it reads as a secondary tools row
  instead of competing with device groups for visual priority.
- Refined folder overlay sizing, spacing, and responsive breakpoints so the
  grouped-device flow remains readable on desktop, tablet, and mobile widths.

## Verification

- Verified Launchpad static regression coverage.
- Verified the frontend production build.
