# New Horizons Desktop v0.4.14

## Changes

### Add experiment pressure-calibration preset

- Added a new **Experiment (10 points)** pressure-calibration preset in Device
  Settings.
- The preset uses the fixed 10-point sequence:
  `4.5, 6, 8, 11, 14, 18, 23, 28, 34, 40` kPa.
- Added matching translations for English, Japanese, and Simplified Chinese.
