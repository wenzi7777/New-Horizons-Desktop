# New Horizons Desktop v0.4.7

## Fixes

### Launchpad device icon labeling

- Replaced the Launchpad `NE` placeholder in device icons with the last four
  MAC characters from each device UID.
- Rendered the MAC suffix as a stable two-line `2 + 2` mark so folder preview
  chips and expanded device cards keep the same compact visual rhythm.
- Added static regression coverage to ensure the Launchpad keeps deriving and
  rendering the MAC-suffix mark from the device UID.

- Verified the Launchpad static regression suite.
- Verified the frontend production build.
