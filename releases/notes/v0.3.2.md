# New Horizons Desktop v0.3.2

## Changes

### Bug Fix: Apply Tare Command

- Fixed `unknown_command` error when clicking **Apply Tare** in the calibration
  panel. The backend command allowlist (`MAINTENANCE_COMMANDS`) was missing
  `calibration_tare_capture`; only the legacy alias `calibration_capture_tare`
  was present. Both names are now accepted.
