# New Horizons Desktop v0.3.6

## Changes

### Localized device wiki delivery

- Added localized `en` / `ja` device wiki trees for the currently documented
  boards.
- `/api/wiki` and `/api/wiki/document` now accept a `lang` query parameter and
  fall back to the device root when a localized directory is unavailable.
- `DeviceWikiPage` now requests wiki listings and markdown documents using the
  active UI locale, then reselects the localized `README.md` automatically.

### Filter command boundary cleanup

- Kept `set_filter` available through backend command routing and terminal/raw
  command surfaces.
- Removed the general Device Settings filter card so the optional scan-time
  filter is no longer exposed as a normal end-user setting.

### OTA and wiki alignment

- Corrected the `VD-CTL/R v2.3.D GCU LTS` default OTA manifest track to
  `arduino-gcu-v23d-lts-latest.json`.
- Updated localized wiki firmware references and OTA examples to the current
  `v0.9.0` firmware release.
