# New Horizons Desktop v0.4.0

## Changes

### Settings taxonomy refresh

- Reorganized Device Settings into task-oriented tabs: Overview, Hardware,
  Runtime, Maintenance, Diagnostics, Files, and Experimental.
- Moved identity, gateway, hardware, runtime, diagnostics, and file controls
  into clearer sections without changing the backend command contract.

### Experimental signal filter controls

- Restored the live-stream Signal Filter controls under the Experimental tab.
- Added explicit controls for enabled state, median window, and low-pass alpha.
- Kept the filter disabled by default when device status does not report a
  filter payload, avoiding stale enabled state across device switches.
- Added copy that calls out the latency tradeoff so the filter is not presented
  as a normal low-latency runtime setting.

### Regression coverage

- Added static coverage for the new settings tab taxonomy.
- Added regression checks that the filter UI remains Experimental-only while
  `set_filter` stays available through the existing command surface.
