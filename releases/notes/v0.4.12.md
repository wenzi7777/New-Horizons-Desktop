# New Horizons Desktop v0.4.12

## Changes

### Locale set cleanup and browser-language default

- Added a complete Simplified Chinese (`zh-CN`) desktop translation alongside
  English and Japanese.
- Simplified the supported desktop locale set so the UI now only exposes:
  English, Japanese, and Simplified Chinese.
- On first launch, the desktop now follows the browser language automatically
  when it matches a supported locale, and falls back to English otherwise.
- The selected locale is still persisted locally, so later manual language
  changes remain sticky across reloads.

## Verification

- Frontend production build passes with the updated locale set and selector.
