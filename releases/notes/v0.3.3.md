# New Horizons Desktop v0.3.3

## Changes

### Pressure Control Plugin

- Removed wrench emoji icon from plugin card and header.
- Added **Reference Sensor** reading (force gauge value) to the live panel
  alongside current kPa.
- Fixed **Stop** button: the system now sets target pressure to 0 kPa before
  releasing control, so the valve closes cleanly instead of leaving the pump
  uncontrolled.

### Safety Prompts

- Added a confirmation dialog before starting pressure in both the Pressure
  Control plugin and the automated calibration panel. The dialog uses a custom
  modal (not the browser's native `window.confirm`).
- Added a dismissible warning banner after pressure stops or calibration
  completes, reminding the operator to turn off the air compressor.
- Replaced ⚠️ emoji icons with `lucide-react` SVG (`TriangleAlert`) for
  consistent rendering across platforms.

### Bug Fix

- Fixed `unknown_command` error when clicking **Apply Tare**: added
  `calibration_tare_capture` to the backend `MAINTENANCE_COMMANDS` allowlist
  (`service.py`).
