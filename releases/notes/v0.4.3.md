# New Horizons Desktop v0.4.3

## Improvements

### Maintenance calibration workspace redesign

- Reorganized the Maintenance page into four clear areas: status summary,
  guided calibration flow, capture/preview workspace, and advanced actions.
- Removed the duplicated flash-usage refresh and file-browser entry points
  from the calibration workspace so file and storage diagnostics stay in the
  dedicated Files section.
- Added step-by-step guidance for entering maintenance mode, starting a draft
  session, capturing tare, capturing calibration levels, committing the
  session, and enabling the finished profile.
- Added explicit disabled-state reasons and an offline warning so operators can
  see why a calibration action is unavailable before clicking.

## Verification

- Added regression coverage for the calibration flow recommendation and
  disabled-reason logic.
- Verified the frontend test script and production build after the redesign.
