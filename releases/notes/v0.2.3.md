# New Horizons Desktop v0.2.3

## Changes

### Pressure Calibration: Manual Confirm UI

- Added a manual pressure calibration confirm dialog that requires explicit user
  confirmation before committing a calibration session, preventing accidental
  overwrites.

### Visualization: Calibrated Level Display

- Sensor matrix cards in the Visualization page now show the calibrated level
  value alongside the raw ADC reading when calibration is active.
