# New Horizons Desktop v0.2.8

## Changes

### Pressure Calibration: Use IMADA Force (N) as Calibration Level

- Reference Pressure Sensor card now displays IMADA force in Newtons (N) instead
  of UNO pressure in kPa, matching the physical quantity being measured.
- `calibration_capture_all` level field is now set to the IMADA N reading; UNO
  kPa value is used as a fallback when the IMADA reading is unavailable.
- Log messages updated to reflect the correct unit from the IMADA device.
