# New Horizons Desktop v0.3.0

## Changes

### Plugins Page: Extensible Tool Framework

- Added a new **Plugins** page (`/plugins`) accessible from the top navigation bar
  (admin role only). The page displays a card grid of available plugin tools;
  clicking a card navigates to `/plugins/:pluginId` for the full plugin view.
- Introduced a `PLUGIN_REGISTRY` pattern in `PluginsPage.tsx`: adding a new plugin
  requires only one entry in the registry array — no changes to routing or
  navigation boilerplate.

### Pressure Control Plugin

- Added the first plugin: **Pressure Control** (`/plugins/pressure-control`).
- Reuses the existing Pressure Cal API credentials (configured in
  Settings → Maintenance → Pressure Calibration or within the plugin itself).
- Flow: enter target kPa (0–45) → Start → system pressurizes via
  `/api/pressure-cal/target` → live pressure bar updates every 500 ms via
  `/api/pressure-cal/readings` → stabilization detection (±0.5 kPa for 5
  consecutive samples) → badge transitions Pressurizing → Stabilizing → Stable →
  operator presses Stop.
- Safety limit enforced: Start is disabled when target > 45 kPa or target < 0.
