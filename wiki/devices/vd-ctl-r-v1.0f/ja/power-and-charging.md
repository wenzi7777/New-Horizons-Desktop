# 電源管理と充電

## 電源状態

| 状態 | CPU 周波数 | 説明 |
|------|-----------|------|
| Normal | 240 MHz | すべてのサブシステムが動作中 |
| SoftOffBattery | 80 MHz | バッテリー駆動でサスペンド、5 秒ごとにライトスリープ |
| SoftOffCharging | 80 MHz | 充電中にサスペンド、2 秒ごとにライトスリープ |

## アクションボタンの動作

| 入力 | 時間 | 動作 |
|------|------|------|
| 短押し | 50〜500 ms | ソフトオフからウェイク；通常モードでは何もしない |
| 長押し | 1500 ms 以上 | 通常時：シャットダウンアニメーション → ソフトオフ；ソフトオフ時：（短押しのみでウェイク可能） |

## ソフトオフモード

ソフトオフは低電力サスペンド状態です。ファームウェアは低い周波数で動作を続けますが、スキャン、WiFi、IMU は一時停止されます。

**ソフトオフへの移行：**
1. Normal モードでアクションボタンを長押しします。
2. `PowerTransitionShutdown` LED アニメーション（白いフェードアウト）が再生されます。
3. アニメーション完了後、充電器の状態に応じて `SoftOffBattery` または `SoftOffCharging` に移行します。

**ソフトオフ中：**
- バッテリー駆動：SK6812 は消灯；デバイスは 5 秒間隔でライトスリープします。
- 充電中：SK6812 は `SoftOffCharging`（オレンジ）または `SoftOffChargeDone`（ティール）を表示；ライトスリープは 2 秒間隔です。

**ソフトオフからのウェイク：**
1. アクションボタンを短押しします。
2. `PowerTransitionWake` LED アニメーション（白いライズ）が再生されます。
3. すべてのサブシステムが再開し、Normal モードに戻ります。

**コマンドによるリモート電源制御：**
```json
{"command": "power_set_state", "protocol": "NHO/Arduino/1", "state": "soft_off"}
```
```json
{"command": "power_set_state", "protocol": "NHO/Arduino/1", "state": "normal"}
```

## 充電

BQ25180 IC がバッテリー充電を管理します。充電状態はステータス LED に反映され、センサーパケットの `battery` フィールドでも報告されます。

### 充電状態

| 状態 | LED | 意味 |
|------|-----|------|
| `NotCharging` | —（ランタイム状態に従う） | バッテリー駆動、充電器未接続 |
| `ChargingOrMissing` | オレンジ Solid | 充電器接続済みで充電中；またはバッテリー未検出 |
| `ChargeDone` | ティール Solid | バッテリー満充電 |

### 充電プロファイル

`set_charge_profile` コマンドで設定します：

| プロファイル | 説明 |
|-------------|------|
| `Balanced` | デフォルト、日常使用に安全 |
| `Fast` | 高電流、発熱あり |
| `Slow` | 控えめな電流、長時間充電に適切 |
| `UltraSlow` | 最小電流、古いセルに最適 |
| `Extreme` | 最大電流—注意して使用 |

```json
{
  "command": "set_charge_profile",
  "protocol": "NHO/Arduino/1",
  "profile": "Balanced"
}
```

### センサーパケット内のバッテリーデータ

パケットヘッダーに `BATTERY_FLAG`（0x02）が設定されている場合、ペイロードに以下が含まれます：

| バイト | フィールド | 説明 |
|--------|-----------|------|
| 0 | `status` | BQ25180 ステータスレジスタ |
| 1 | `fault` | BQ25180 フォルトレジスタ |
| 2–3 | `vbat_mv` | バッテリー電圧（ミリボルト、リトルエンディアン uint16） |
